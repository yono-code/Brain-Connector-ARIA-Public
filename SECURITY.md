# Security Policy

## Supported Versions

Security fixes are currently provided for:

- `0.1.x`

## Reporting a Vulnerability

Please report vulnerabilities privately to:

- `yonogames.dev@gmail.com`

Do not open a public issue for unpatched vulnerabilities.

## What to Include

- Affected version (`package.json` version or commit SHA)
- Reproduction steps
- Impact summary
- Any proof-of-concept details needed to validate the issue

Please avoid sending secrets, personal data, or private keys in the report.

## Response Targets

- Acknowledgement: within 24 hours
- Initial triage response: within 3 business days

## Scope Note

Brain Connector ARIA is currently maintained as a local-only release line.
Network/sync features are out of scope unless explicitly reintroduced through a separate review track.
